📋 AACOMAS Application Guide

What is AACOMAS?

The centralized application for DO schools, like AMCAS is for MD schools. You CAN and SHOULD apply to both simultaneously. Thousands of applicants do this every year.

Key Differences from AMCAS

  • • Different GPA calculation method
  • • Personal statement should reflect osteopathic philosophy
  • • Separate transcript submission process
  • • $198 first school + $60 per additional
